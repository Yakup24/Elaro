# Elaro v0.2.2

Release date: 2026-05-12

## Added

- API integration coverage for the public health endpoint and security response headers.
- CI-published API test result and coverage artifacts.
- Tag-driven GitHub Release automation for public release pages.

## Changed

- API test suite now covers both customer and admin JWT claims.
- JWT configuration tests now verify that weak signing keys fail fast with a clear error.

## Verification

```bash
dotnet test ElaroAPI/ElaroApi.sln -c Release --collect:"XPlat Code Coverage"
```

The release workflow runs automatically when a `v*` tag is pushed.
